# VitaTrack - Health & Lifestyle App

## 📱 Description
VitaTrack is a Flutter-based health tracking application that helps users monitor their daily nutrition, physical activity, and overall wellness.

## 🎯 Features

### 🍽 Meal Tracker
- Track food intake
- Calculates calories, vitamins, and minerals

### 🚶 Activity Tracker
- Tracks steps
- Calculates calories burned

### 📊 Health Report
- Displays health score
- Shows calories (eaten vs burned)
- Step progress with indicators

### 🧠 Smart Analysis
- Balance Index
- Day Type (Healthy / Inactive / Overloaded)
- Energy Level
- Personalized feedback

### 📈 Daily Comparison
- Compare today vs yesterday
- Shows increase/decrease in:
  - Steps
  - Calories
  - Health Score

### ⚙️ Settings
- Dark mode toggle
- Profile section
- About page
- Logout functionality

## 🛠 Technologies Used
- Flutter
- Dart

## 🎯 Objective
To promote a healthy lifestyle by combining nutrition tracking, activity monitoring, and smart analysis in a simple and user-friendly interface.
